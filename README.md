# Coming Soon One-Pager

A high-performance, responsive, privacy-first "Coming Soon" landing page designed to run on the Cloudflare edge. Built with vanilla HTML5/CSS3 and integrated with Cloudflare Pages, Pages Functions (Workers), and Cloudflare Web Analytics.
